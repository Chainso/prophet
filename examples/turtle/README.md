# Turtle Examples

These examples focus on Prophet's Turtle target (`gen/turtle/ontology.ttl`) and SHACL validation against the base metamodel in `prophet.ttl`.

Available examples:
- [Minimal Support Domain](prophet_example_turtle_minimal/README.md)
- [Small Business Domain (Complex)](prophet_example_turtle_small_business/README.md)

Both examples use:
- `generation.targets: [turtle, manifest]`
- deterministic output settings
- SHACL validation with `pyshacl`
